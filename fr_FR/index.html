Acceptez les paiements Bitcoin avec 0% de frais et sans intermédiaire.

BTCPay Server est un processeur de paiement pour crypto-monnaies auto-hébergé et à code source ouvert. C'est sécurisé, anti-censure et gratuit.

Recevez vos paiements en Bitcoin et altcoin sans frais ou intervention d'un intermédiaire. Vous êtes votre propre processeur de paiement. Les fonds vont directement dans votre portefeuille; votre clé privée n'est jamais requise.
